# swipemenulistview
带侧滑菜单的ListView

根据[https://github.com/baoyongzhang/SwipeMenuListView](https://github.com/baoyongzhang/SwipeMenuListView)修改而来。

修改内容：

- 有条目菜单打开时，触摸任意位置关闭条目。
- 优化手势加速开启、关闭条目菜单。
- 其他一些小修改

## 使用方法
在module的build.gradle添加依赖：

	dependencies {
		compile 'com.github.fszeng2011:treeadapter:1.0.0'
	}
